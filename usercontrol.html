User Control and Freedom

Users often choose system functions by mistake and will need a clearly marked "emergency exit" to leave the unwanted state without having to go through an extended dialogue. Support undo and redo.

img

The website is good at allowing the user to freely roam and be able to go back to their previous page, displaying a path on the top of each section for users to click on. For example, on (http://www2.le.ac.uk/research/current-research/global). Also, the user can always select the back button to go back to their previous page. Almost all of the interaction in the website does not open into a new window or tab. Every clickable link that does not lead to a social media website or email loads on the same tab, and therefore makes it easy to navigate back and forth.
