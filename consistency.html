Consistency and Standards

Users should not have to wonder whether different words, situations, or actions mean the same thing. Follow platform conventions.

Users are provided with an interactive map that succeeds at providing an organized directory, but fails to hold a consistent system standard. Upon navigating to the “Maps and Directions” page in particular, the user is bombarded with external hotlinks to outside gps mapping services (Google Maps) breaking sustained attention and minesweeping users off the site.

The system is substantially organized for a website home to a large source of university information, but the search tools are not consistent with universal web browsing standards. The “News and Events” repository includes an additional search tool that searches not only the standard site directory, but also a specific library of system information -- news and events. Additional affordances are introduced to users that are simply seeking specific information, but now must decide between different tools seemingly suggesting the similar functionalities.
